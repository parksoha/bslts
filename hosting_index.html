현재 페이지가 보여지는 것은 "고객님의 계정이 정상적으로 세팅되었다는 것"을
뜻합니다.
 
계정 사용에 대한 보다 자세한 사항은 세팅 시 발송되는 세팅확인 메일을 확인하시기
바랍니다.
세팅확인 메일은 관리자 E-mail(한메일제외)주소로 발송됩니다.
따라서 회원정보와 관리자 정보는 항상사용가능한 정보로 업데이트 해주시기 바랍니다.
 
[호스팅 사용 안내]
 
1.WEB 접속 안내
① WEB 접속은 고객님의 도메인으로 접속가능합니다.사이트 접속이 정상적으로 이루
    어지지 않을시에는 도메인네임서버가 정상적으로 세팅되어 있는지를 확인하십시오.
② 네임서버를 변경하셨을때는 변경 후,약24시간이 지나야 정상적인접속이 가능합니다.
③ 무료 도메인을 통해 접속할 경우에는 30분 이내 접속이 가능합니다.(단, 신청한경우)
 
2. FTP/TELNET 접속안내
① 접속 주소는 고객님의 도메인 입니다.
② 접속 아이디는 고객니의 계정 아이디입니다.
③ 접속 패스워드는 고객님 계정의 패스워드입니다.
 
3. 주의사항
① 서비스할 홈페이지 파일들은 www 폴더 아래에 두십시오.
    www를 삭제하면 홈페이지 서비스가 불가하오니 삭제하지 마시고,혹시 삭제된 경우
    새로 만드시기 바랍니다.
② 홈페이지 메인페이지로 사용되는 파일은 반드시 www폴더 아래에 있어야 하며 파일
    이름은 index.html(htm, php 등)이어야 합니다.
③ 고객님의 계정관리에 필요한 기능들은 대부분 호스팅 메뉴의 나의서비스관리를 통해
    가능합니다.
